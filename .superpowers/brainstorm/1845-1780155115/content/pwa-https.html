PWA 与 HTTPS 的关系

分清楚"要不要证书"

各功能对 HTTPS 的要求

🔴
Service Worker
必须 HTTPS（localhost 开发时除外）
🟡
安装到桌面（manifest.json）
建议 HTTPS，但 manifest 本身走 HTTP 也能加载
不过浏览器安装提示需要 HTTPS + Service Worker 同时满足
🟢
你的全部功能（React + Tailwind + localStorage + API 调用）
不需要证书，HTTP 完全够用

结论

既然我们跳过 Service Worker，证书不是必需的。

✅ 纯 HTTP 就能开发部署
✅ manifest.json 纯装饰作用（桌面图标 + 启动画面）
✅ 所有功能正常使用
